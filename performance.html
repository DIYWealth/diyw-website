Portfolio Selection

The test of a portfolio strategy is in its performance. The performance can vary based on a number of factors. On this page, we provide tools to uncover two important considerations for a portfolio (1) the portfolio size and (2) the market capitalisation (MCAP) of each company included in the portfolio.

Portfolio Size

Select from the dropdown menu below the number of stocks in the portfolio.

Minimum market capitalisation (MCAP)

For each stock in the portfolio, select from the dropdown menu below the minimum MCAP ($USD) of each company.
